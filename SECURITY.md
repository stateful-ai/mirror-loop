# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Mirror Loop, please open a private
security advisory on GitHub (Security > Advisories > New draft) rather than
filing a public issue.

We aim to acknowledge reports within a few days and to publish a fix or
workaround as quickly as the severity warrants.

## Scope

Mirror Loop is a research/prototype project. Authentication, multi-tenant
isolation, and exposed network services are out of scope.
