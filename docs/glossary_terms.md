# Glossary terms (acronyms only)

Reverse-lookup for the acronyms and short names sprinkled through the
code and docs. Append to this file when you introduce a new one.

- **CLI** — Command-line interface.
- **CI** — Continuous integration.
- **PR** — Pull request.
- **YAML** — Yet Another Markup Language (config format).
- **JSON** — JavaScript Object Notation (machine-readable serialization).
- **MVP** — Minimum viable product.
- **TBD** — To be decided.
