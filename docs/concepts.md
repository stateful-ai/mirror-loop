# Concepts

## Session
A session is a single instance of a user playing the game. It includes all interactions and adaptations made during the gameplay.

## Adaptation
Adaptation refers to the dynamic changes made to the game experience based on the player's actions and choices. This ensures a personalized and evolving gameplay.

## Reflection
Reflection is a key moment in the game where the player receives feedback on their choices and actions. It helps guide the player towards certain outcomes and provides insights into the game's mechanics.

## Variant
A variant represents a different configuration or version of the game. Variants can be used to test different gameplay scenarios or to provide alternative experiences for the player.

## Loop
The loop is the core gameplay cycle that repeats throughout the session. It involves a series of beats, choices, and feedback mechanisms designed to engage the player and drive the narrative forward.
