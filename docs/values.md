# Values

Decisions we've already made about what we'll bias toward.

## Craft over scale
We aim for *taste, craft, and delight* — not user counts.

## Determinism over speed
We pay the determinism tax willingly. Adaptive features that drift
run-to-run aren't features.

## Templates over generation
Templated text is the default. The LLM enters only when the template
fails to carry the moment.

## Founder attention is the bottleneck
Anything that protects that — fewer dashboards, fewer notifications,
one-page briefings — wins by default.
