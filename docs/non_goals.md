# Non-goals

Things Mirror Loop deliberately is *not*. Listed here so they can be
declined quickly without re-relitigating each time they come up.

- **Not a chatbot.** The Mirror observes; it does not converse with the
  player in free-form natural language.
- **Not a procedural-content engine.** Scenes are authored; the
  adaptation is in *ordering and emphasis*, not in generating new prose.
- **Not a research framework.** We are not building a generic
  behaviour-modeling library; everything we add must serve this game.
- **Not multiplayer.** No shared state across players; no leaderboards.
- **Not analytics-driven.** No retention, DAU, or session-length goals.
  Build what would surprise and delight the founder and stop there.
