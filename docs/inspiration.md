# Inspiration

Aesthetic touchstones — references the project's tone should resonate
with, not imitate. Add a bullet whenever a piece of writing, code,
game, or design influences how you think about the project.

## Tone
- Annie Dillard's prose: observational, present-tense, low ornament.
- Werner Herzog's voice-over: declarative, never self-important.

## Code
- The Rust compiler's error messages: when they tell you what's wrong,
  they tell you the next move too.
