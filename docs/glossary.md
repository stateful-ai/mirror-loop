# Glossary

- **Mirror axis** — the caution ↔ aggression dimension the system measures and adapts along.
- **Intake** — the lab questionnaire that opens the run and seeds the player model.
- **Recalibration** — the loop 3 beat where the Mirror surfaces what it noticed.
- **Reflection beat** — the one-sentence observation paired with an in-fiction evidence quote.
- **Adaptive arm** — the run mode where the system re-orders scenes; opposite of the **baseline arm**.
- **Adaptive prediction** — the system's bet on the next choice, made visible via re-ordering.
- **Five-loop spine** — the canonical Act 1 cadence (loops 1-5).

Add new terms here as design discussions converge on shared vocabulary.
