# Intent

This project exists to find out whether a particular shape of design
works — not to ship a polished product.

The bet:
- A deterministic core paired with thin adaptive seams produces a more
  interesting experience than monolithic ML or pure scripted content.
- Templated text first; LLM is the surprise, not the default.
- Founder attention is the bottleneck; small one-page tools beat
  dashboards.

If those bets prove wrong, the codebase should still be readable —
so the next attempt starts from honest scaffolding.
